UpdateProductLimits (C#)
===============================================================================

Overview
-------------------------------------------------------------------------------

This example demonstrates using the TT US API to update product limits for an account

Instructions
-------------------------------------------------------------------------------

1. Type TTUS username and password and click connect.
2. Select the account to update product limits on.
3. Populate appropriate fields
3. Click update button and look for status message.


TT US API Objects
-------------------------------------------------------------------------------

AccountGroup
AccountGroupProfile
ProductLimit
ProductLimitProfile
Gateway
